# 🚀 ¿Cómo realizar la entrega?

1. Ingresen al repositorio del trabajo.
2. Realicen un **Fork** para crear una copia en su cuenta de GitHub.
3. Completen todas las consignas del trabajo práctico sobre esa copia.
4. Realicen los **commits** que consideren necesarios durante el desarrollo.
5. Suban los cambios a su repositorio de GitHub.
