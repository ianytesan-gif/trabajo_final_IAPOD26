# 🚀 ¿Cómo realizar el trabajo?

## Paso 1. Crear un Fork

Ingresen al repositorio de la cátedra y presionen el botón **Fork**.

Esto creará una copia del trabajo en su propia cuenta de GitHub.

⚠️ **Todo el trabajo deberá realizarse sobre esa copia.**

---

## Paso 2. Clonar el repositorio

Abran Git Bash o la terminal y ejecuten:

```bash
git clone https://github.com/TU_USUARIO/trabajo_final_IAPOD26.git
```

Reemplacen **TU_USUARIO** por su nombre de usuario de GitHub.

Luego ingresen a la carpeta:

```bash
cd trabajo_final_IAPOD26
```

---

## Paso 3. Abrir el proyecto

Pueden trabajar desde:

- Google Colab
- Visual Studio Code
- Jupyter Notebook

Completen todas las consignas respetando la estructura original del trabajo.

---

## Paso 4. Guardar los cambios en Git

Cada vez que finalicen una parte importante del trabajo ejecuten:

```bash
git add .
```

Luego creen un commit:

```bash
git commit -m "Resuelvo la consigna 1"
```

Finalmente suban los cambios a GitHub:

```bash
git push origin main
```

Repitan este proceso tantas veces como sea necesario durante el desarrollo.

# 💻 Comandos básicos de Git

## Clonar el repositorio

```bash
git clone https://github.com/TU_USUARIO/trabajo_final_IAPOD26.git
```

## Ingresar a la carpeta

```bash
cd trabajo_final_IAPOD26
```

## Ver el estado del proyecto

```bash
git status
```

## Agregar cambios

```bash
git add .
```

## Crear un commit

```bash
git commit -m "Descripción del cambio"
```

Ejemplos:

```bash
git commit -m "Resuelvo la consigna 1"

git commit -m "Agrego gráficos"

git commit -m "Corrijo errores"

git commit -m "Finalizo el trabajo"
```

## Subir los cambios

```bash
git push origin main
```

## Descargar cambios (si fuera necesario)

```bash
git pull
```
